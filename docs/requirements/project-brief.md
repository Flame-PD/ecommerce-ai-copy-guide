# 项目需求简述

## 项目题目

电商 AI 商品文案生成与智能导购助手。

## 目标用户

- 商品运营：希望快速生成商品标题、卖点、详情页文案和广告语。
- 电商客服 / 导购：希望根据用户需求给出可解释的推荐建议。
- 商品负责人：希望从评论中发现高频好评点、差评痛点和优化方向。
- 直播运营：希望快速生成直播口播流程、互动问题和转化话术。

## 核心功能

1. 商品文案智能生成：输入商品名称、目标人群、语气和卖点，输出标题、卖点、详情页文案和广告语。
2. 智能导购与推荐问答：输入用户需求、预算和候选商品，输出首推商品、推荐理由、备选商品和购买建议。
3. 用户评论情感分析：输入评论列表，输出正中负情绪统计、关键词、痛点和优化建议。
4. 直播 / 短视频脚本自动生成：输入商品、时长、语气和亮点，输出分段脚本与互动问题。

## 非功能要求

- 支持 Docker 化部署，基础设施包含 PostgreSQL 与 Redis。
- 使用 Git 分支和 PR 管理协作。
- 后端优先采用 Flask，接口使用结构化请求/响应模型。
- 前端使用 Vue 3 + Vite，优先保证课程演示可用。
- AI 接口需要可替换，开发初期允许使用 Mock Service 稳定联调。
