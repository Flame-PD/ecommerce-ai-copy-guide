# 初始化后续计划

## Milestone 1：初始化骨架

- 完成 README、需求简述、架构笔记和计划文档。
- 完成 Flask API、Pydantic 请求模型和 Mock AI Service。
- 完成 Vue 演示首页和后端状态展示。
- 完成基础测试和构建验证。

## Milestone 2：数据模型与持久化

- 设计商品、评论、生成任务、推荐记录和会话表。
- 接入 PostgreSQL，并补充数据库迁移流程。
- 将 Mock 结果保存为可展示的历史记录。
- 使用 Redis 缓存热点商品和短期会话上下文。

## Milestone 3：AI 能力接入

- 选择模型供应商和 API 形式。
- 将 Mock Service 抽象为 provider interface。
- 沉淀文案、导购、情感分析和直播脚本提示词模板。
- 增加失败兜底、超时控制和结果质量检查。

## Milestone 4：展示与汇报

- 增加管理后台或数据看板入口。
- 准备商品样本、评论样本和演示流程。
- 输出架构图、业务流程图和最终汇报材料。
- 整理部署说明和小组分工记录。
